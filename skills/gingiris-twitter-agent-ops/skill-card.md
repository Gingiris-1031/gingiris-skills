## Description: <br>
This skill gives an AI agent a Twitter/X ghostwriting and account-operations SOP for persona calibration, source-backed drafting, scheduling, pre-publish checks, tweet-log tracking, and weekly performance reports. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Creators, marketers, and account operators use this skill to turn an AI agent into a supervised Twitter/X ghostwriter that drafts on-brand posts, maintains a source index, prepares weekly schedules, checks content before publishing, and summarizes performance. It is intended for accounts the user is authorized to manage. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill can support live-account posting workflows without built-in human approval, privacy, or platform-compliance guardrails. <br>
Mitigation: Use it in draft mode or require explicit approval before every post, connect only accounts the user is authorized to manage, and follow applicable Twitter/X account and content policies. <br>
Risk: The workflow encourages indexing source content that could include private, confidential, copyrighted, or sensitive material. <br>
Mitigation: Index only material the user has rights and consent to use, avoid sensitive or confidential inputs unless a retention plan exists, and remove sources that should not be reused. <br>
Risk: Drafts or analytics can include misleading claims if source data, performance figures, or dynamic numbers are stale or unverified. <br>
Mitigation: Require every concrete number to map to a traceable source, refresh dynamic values before publishing, and review each draft with the pre-publish checklist. <br>
